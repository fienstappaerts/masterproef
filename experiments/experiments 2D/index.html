    These experiments in 2D where the very first images I created with a system. From these I went ahead and gotten inspired to go ahead and create 3D structures.

    I decided to make lots of work on this part of my project. There are more complicated structures but also more simple images.

    In the system there is a command 'lsys', lsys was originally inspired by the idea of l-system. This is a collection of production rules that expand each symbol into some larger string of symbols. The command at this moment isn't recursive anymore but went more to the idea of turtle graphics, where a structure is created and guided step by step.
